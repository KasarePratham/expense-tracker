# Daily Expense Tracker

A beginner-friendly Python project to keep track of your day-to-day expenses directly from the terminal.  
You can add expenses, view all saved records, calculate your total spending, and even check how much money you spent in each category.

---

##  Features

- Add new expenses with:
  - Date
  - Category
  - Short note/description
  - Amount spent
- View all saved expenses in a clean format
- Calculate total expenses instantly
- See category-wise spending summary
- Simple menu-driven console interface
- Basic input validation for smoother usage

---

## What I Learned From This Project

This project helped me practice important Python basics such as:

- Taking user input using `input()`
- Displaying output with `print()`
- Using `if-elif-else` statements for menu handling
- Storing data using:
  - `list`
  - `dictionary`
- Using loops to process records
- Calculating totals and summaries
- Formatting output using f-strings
